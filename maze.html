The Maze

An Interactive puzzle game

Task

Game Design, Interaction Design, 3D Modeling, Programming, Sound Effect

Date

FEB 2016 - JUL 2016

Category

Game, Interaction

Project Description

The Maze is a PC-based interactive puzzle game, the idea came from an old video game “Tower of the Sorcerer”.


Storyline

Our storyline is quite simple and dark. The princess is caputured by the devil and she is put in an seemingly endless maze. Each storey there is a "transfering point" and it will transfer the princess to an unknown storey. The princess have 4 kinds of magic: ice, flame, flying and rock so she can deal with some special situations. The end of the game is that the princess finds the prince and also find out the prince is the devil who captures her. Then the princess will be trapped in the maze forever.

Interaction Methods

Moving of the character can be controlled by input of keyboard. In order to add some fun into the game, we make every storey to be a 3*3 square. Then we use dancing mat connected with Arduino for controlling the movement of the character. Also, we add a hard mode for the game, which is, there will be partly dark, falling rocks, longer cold times... and user can choose to turn these on or not.

Technology

Game Engine: Unity3D

Programming Language: C#, JavaScript

External Devices: Dancing Mat, Arduino

Modeling Tool: Maya

Sound Effect Software: Adobe Audition
