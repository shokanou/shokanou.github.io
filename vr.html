Interaction Method Design

In order to create a better user experience, we use eye-tracking technology to controll in the game. If the obeject is gazed at for more than 5 seconds, it will be smashed also with effects thus the user can relieve pressure. Also, user can controll movement in the game by game gear from Baofeng Glasses.
