Project Description

PulseMe is a project of course "Physical Interaction Design and Realization". The theme of our project is "Shape Shifting", which means the output interaction shoule be related to shape changing. We prototyped an armband which targeting at elderly people: it measures real-time blood pressure, which the pressure reaches some limit, the armband will shrink and reverse thus create a "robotic touch" to notify user to take care of body condition.
